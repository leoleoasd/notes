\begin{outline}[enumerate]
    \1[4.] 下列各式成立吗？请证明你的结论。
        \2[(3)] $rs = sr$
        
        不成立。如：令$r = a,\;s = b$。则$rs$识别的语言为$\set{ab}$，$sr$
        识别的语言为$\set{ba}$。
        \2[(6)] $(r+s)^* = r^* + s^*$

        不成立。如：令$r = a,\;s = b$。则$rs$识别的语言为$a,b$组成的串，$sr$
        识别的语言为$a$组成的串或$b$组成的串。
\end{outline}
